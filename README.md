# Credit-card-fraud-detection

โปรเจกต์ Machine Learning สำหรับตรวจจับธุรกรรมบัตรเครดิตที่มีความเสี่ยงในการทุจริต (Fraud Detection) บนข้อมูลที่มี **class imbalance สูง (fraud < 0.2%)**

## Methods

* Data preprocessing & feature scaling
* Handling imbalanced data ด้วย **class weighting**
* Models: Logistic Regression, HistGradientBoosting
* Evaluation: ROC-AUC

## Results

* **Logistic Regression:** ROC-AUC = **0.95**
* HistGradientBoosting: ROC-AUC = 0.79

## Tools

Python, Pandas, NumPy, Scikit-learn
